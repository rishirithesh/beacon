# Beacon Product Roadmap

Version: 1.0

---

# Vision

Beacon aims to become the universal investigation platform for developers.

It should eventually investigate failures occurring in

- Editors
- Terminals
- CI/CD Pipelines
- Docker
- Kubernetes
- Cloud Platforms
- Git
- Production Logs

without changing the Investigation Engine.

---

# Guiding Principles

Every release should satisfy one of the following.

✓ Improve investigation accuracy

✓ Reduce investigation time

✓ Reduce AI usage

✓ Increase supported ecosystems

✓ Improve developer experience

Never add features that do not support these goals.

# What Beacon Will Never Become

Beacon will never become

A chatbot.

A code editor.

A replacement for VS Code.

A general-purpose AI assistant.

Its purpose is singular.

Understand the cause,

not the symptom.
